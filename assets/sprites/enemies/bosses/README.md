# Boss Sprites

Place boss spritesheets here. Bosses are typically larger than regular enemies.

## Recommended Files
- `flame_lord.png` - Fire boss (32x32)
- `frost_queen.png` - Ice boss (32x32)
- `storm_titan.png` - Lightning boss (48x48)

## Animation Rows
- **Row 0**: Idle (4-6 frames)
- **Row 1**: Move (4 frames)
- **Row 2**: Attack 1 (4-6 frames)
- **Row 3**: Attack 2 (4-6 frames)
- **Row 4**: Special attack (6-8 frames)
- **Row 5**: Phase transition (4 frames)
- **Row 6**: Hurt (2 frames)
- **Row 7**: Death (6-8 frames)
